\section*{Overview}
This document provides the complete set of optimized parameter values $(\beta_c, \beta_h, \omega_c, \omega_h)$, maximum achievable efficiency $\eta_{\mathrm{max}}$, and corresponding work output $W_{\mathrm{ext}}$ for each configuration (Case 1 to Case 4) as a function of the anharmonicity parameter $\alpha \in [0.00, 0.20]$. These parameter values correspond directly to the curves plotted in \texttt{eta\_vs\_alpha\_all\_cases.pdf}.

\subsection*{Physical Constraints Enforced}
All optimizations enforce strict thermodynamic and platform boundaries:
\begin{enumerate}
    \item \textbf{Platform Ratio Caps:} $\frac{\omega_h}{\omega_c} \le 7.0$ and $\frac{\beta_c}{\beta_h} \le 25.0$ (based on diamond NV-centre platform caps).
    \item \textbf{Perturbation Validity:} $\varepsilon_c \le 0.10$ and $\varepsilon_h \le 0.10$, where $\varepsilon = \frac{3 \alpha \coth(\beta \omega / 2)}{4 \omega^3}$.
    \item \textbf{Engine Operation Condition:} $\beta_c \omega_c > \beta_h \omega_h$ and $W_{\mathrm{ext}} > 0$.
\end{enumerate}
